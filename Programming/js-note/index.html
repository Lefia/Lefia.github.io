變數宣告

命名規則

  1. 變數名必須以英文字母, _, $ 開頭
  2. 變數名可以包含英文字母, _, $及數字
  3. 不可以使用 javascript 的關鍵字、保留字作為變數名稱

變數值類型

  • 原始值
  • 引用值

數組(array)、物件(object)、函式(function)、日期(date)、正規表達式(RegExp)…

基本規則

  • 語句以分號 ; 結尾
  • 當一個 script 區塊發生錯誤時，不會影響到另一個區塊

運算子

  • 任何類型的數據加上字串都等於字串
  • 指定運算子=優先度最低，()優先度最高
