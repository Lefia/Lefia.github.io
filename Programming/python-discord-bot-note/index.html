新增 Discord Application

網址： https://discord.com/developers/applications

  1. 在新增 Application 後，進入設定畫面
  2. 在 Bot 頁面點選 Add Bot
  3. 找到 Bot Permissions 後，勾選 Administrator
  4. 進到 OAuth2 頁面，在 SCOPES 勾選 bot
  5. 複製下面產生的網址，貼在瀏覽器的網址列，將機器人邀請至伺服器

在伺服器需要有相對應的權限才能將機器人邀請進去
