HTML templates + Github

Posted on 2020-09-15 Edited on 2021-02-10 In Website Views:

用最基礎的HTML模板和簡單容易上手的Github Desktop來架設網站

本教學以 Windows 為主

事前準備 & Github Desktop 設置

  1. 進入 Github 的網站，點擊 Sign up 註冊帳號

  1. 註冊完後，重新登入

  1. 進入主畫面，點選 NEW 創建新的資料庫 ( repository )

  1. 依照下圖步驟操作

  1. 開啟新視窗，安裝 Github Desktop

  1. 安裝完成後，回到 Github，點選 Set up in Desktop

  1. 依照下圖步驟操作

下載 html 模板

  1. 進入 TEMPLATED 的首頁

  1. 選擇自己喜歡的模板 ( 右下角有 “Responsive HTML5 Template” 能夠將頁面自動調整至符合裝置的大小 )

    點擊 DEMO 預覽頁面的樣式，點擊 DOWNLOAD 下載模板壓縮檔

  1. 將下載的模板解壓縮後，放到 步驟8 所選擇的資料夾中

上傳至上傳至 Github

  1. 回到 Github Desktop 中，可以發現左手邊的 Changes 多了很多檔案。依照下圖步驟操作，將檔案上傳至 Github

  1. 回到 Github ，重新整理後，可以看見剛才上傳的檔案

  1. 在瀏覽器輸入 https://使用者名稱.github.io 後按 enter，就能看見自己的網站了

上傳之後，要過一段時間才看的見修改的內容
